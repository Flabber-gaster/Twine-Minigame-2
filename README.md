# Twine-Minigame-2
## Devlog
For my game, the idea for the plot came from experiences I had, and I thought it'd work well for this type of game. Because the story is a bit short though, I tried to use the looping mechanic a couple times where It'd make sense. For example, the player loops back to checking the fridge until they choose to take the pack of ham. There were some other ideas I had for looping or branching paths, but because of time constraints, I decided to make the options a bit more simple.
